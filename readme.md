### How to start

(Optional) Make sure you have valid node.js version

```shell
nvm use
```

Or go to [nvm repo](https://github.com/nvm-sh/nvm) to dowload this package

**Install dependencies:**

```shell
npm i
```

**Run local dev server:**

```shell
npm run dev
```
